# FLABBY BIRB

FLABBY BIRB is a recreation of the 2013 mobile game "Flappy Bird" made with Unity.

## Installation

1. Download the ZIP file of the repository and extract the file.

2. Download Unity and Unity Hub from Unity's website:
```
https://unity.com/download
```
3. In Unity Hub, click on Add, Add project from disk, and then click on the extracted folder called "FLABBY-BIRB-main".

4. In the Unity Game Engine, go to File, and click "Build and Run"

5. The game will start running as an application.

## Gameplay

Use the mouse to click on buttons, and the up arrow to "flap up" the bird.
